3 - External Power
[ ] 3.1 - Turn on BAT 1 and BAT 2 switches
[ ] 3.2 - Call external power (Aircraft > Ground Services > External Power)
[ ] 3.3 - Check EXT PWR light is AVAIL (Green)
[ ] 3.4 - Press EXT PWR switch (Light changes to ON Blue)

4 - Air Data Inertial Reference System (ADIRS)
[ ] 4.1 - Turn IR1, IR2 and IR3 selectors to NAV
[ ] 4.2 - Check ON BAT light illuminates briefly and then turns OFF

5 - Auxiliary Power Unit (APU) & Cabin Comfort
[ ] 5.1 - Turn on APU MASTER switch
[ ] 5.2 - Press APU START switch (White ON light appears)
[ ] 5.3 - Wait for green AVAIL light on APU START switch
[ ] 5.4 - Turn on APU BLEED switch (Pneumatic air active)
[ ] 5.5 - Check X-BLEED is set to AUTO
[ ] 5.6 - Check PACK 1 and PACK 2 are ON (Lights OFF on panel = Active)
[ ] 5.7 - Check HOT AIR switch is ON (Light OFF on panel = Active)
[ ] 5.8 - Set PACK FLOW to NORM (or HI if the climate is too hot)
[ ] 5.9 - Adjust COCKPIT, FWD CABIN, and AFT CABIN selectors to your preference (12 o'clock = 24°C)
[ ] 5.10 - Press EXT POWER switch to turn it OFF (Light changes back to AVAIL Green)
[ ] 5.11 - Dismiss External Power (Aircraft > Ground Services > External Power)
